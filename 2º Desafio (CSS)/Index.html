Neste site iremos aprender sobre o que é CSS, sua importância, evoluções, conceito, responsividade e acessibilidade,
seletores, animações e por ultimo e não menos importante melhores práticas para escrever um código CSS!


O que é CSS? E sua importância?

CSS, ou Cascading Style Sheets é reconhecida como uma linguagem de estilo, ela é utilizada por desenvolvedores para dar aparência e efeitos
a páginas de sites e aplicativos construídas sobre uma linguagem de marcação, como o HTML. Com a linguagem, os programadores podem definir e modificar o layout usando:
Fontes tipográficas, tamanho e expressividade dos elementos, posição dos elementos na página, efeitos visuais de movimento, dentre outras possibilidades.

Sendo fundamental para o desenvolvimento web, o CSS garante o padrão da identidade visual de sua página ou aplicativo, tornando-os mais atraentes, eficientes, acessível e alinhados com a identidade visual da marca.


Evolução do CSS:

Inicialmente CSS era limitado utilizado apenas para formatação básica oferecendo um layout simples.
Ao longo dos anos e à medida em que as páginas web recebiam mais complexidade houve maiores demanda por designs mais ricos surgindo a necessidade de uma linguagem de estilo mais robusta. Assim veio a chegada do CSS2, incluindo posicionamento, controle aprimorado sobre layout e a introdução de tipos de mídia para diferentes dispositivos.

E outras versões onde resultaram em diversas melhorias como: maior estabilidade e compatibilidade com navegadores.
Atualmente CSS3 proporciona animações, gradientes e layouts de caixa flexíveis e acreditasse no desenvolvimento futuro do CSS4 poderá incluir
design responsivos, fontes variáveis e sistemas de grade aprimorados.


Principais conceitos e sua influência:

Os conceitos são definidos por:

- Ser responsável pelo visual do site, permitindo a adição de estilos, cores, fontes e outros elementos visuais a uma página da web.

- Dividir o conteúdo (HTML) da apresentação visual, facilitando a manutenção e a personalização do design.

- Permitir a criação de diferentes layouts, incluindo tabelas, grids e ajustes de imagens.

- Garante um padrão de formatação em diferentes navegadores e melhora a responsividade do design, adaptando-o a diversos tamanhos de tela.

Esses conceitos influenciam diretamente na estilização de páginas web porque determinam como os elementos visuais são organizados,
apresentados e adaptados em diferentes dispositivos. Criando uma identidade visual ao site e uma experiência mais agradável para o usuário.


Responsividade e Acessibilidade:

A responsividade CSS é a capacidade de uma página web se adaptar ao tamanho do dispositivo em que é exibida,
garantindo uma experiência agradável tanto para usuários de desktop quanto para aqueles em tablets e smartphones.

No caso de responsividade, o CSS oferece diversas técnicas e ferramentas para tornar uma página web responsiva,
como: Media Queries, que permite aplicar estilos diferentes com base no tamanho da tela, Grid Layout que cede com que crie layouts mais complexos,
permitindo o reposicionamento automático de elementos e entre outros.


Enquanto a acessibilidade CSS, é a possibilidade de um site disponibilizar inclusão aqueles que possuem deficiências visuais, motoras ou cognitiva,
possibilitando com que acessem o conteúdo confortavelmente. Proporcionando ao usuário uma navegação mais inclusiva e acessível.

Basicamente o CSS disponibiliza com que você faça ajustes no seu site, por exemplo, ajustando contrastes de cores, permitindo fontes escaláveis,
destacando elementos navegáveis e reduzindo animações que podem dificultar a navegação.


Seletores de classe, ID e elementos:

Seletores de classe:

No caso de seletores de classe, no CSS são usados para aplicar estilos a um ou mais elementos HTML que queremos estilizar em nossa página web.
Entre tanto as classes permitem estilizar apenas os elementos que possuem um determinado nome de classe, oferecendo mais flexibilidade na personalização do layout.

Aqui, apenas os elementos com a classe "titulo" receberiam o estilo (Terá a cor azul com o fundo amarelo).


Seletores de elementos:

Em CSS, os seletores de elemento são usados para estilizar todos os elementos de um tipo específico em uma página web.
Eles aplicam os estilos diretamente a todas as ocorrências da tag HTML escolhida.

Nesse exemplo a estilização será aplicada em todos os elementos de parágrafo do documento (todos se tornarão vermelho), dificultando casos em que queremos estilizar apenas um elemento específico.


Seletores de ID:

Seletores de ID são utilizados, quando desejamos identificar um elemento específico no nosso documento, para sua representação no CSS atribuímos o “#”.
O seletor de ID estiliza apenas o elemento específico que tem aquele ID, garantindo que o estilo será aplicado de forma única e precisa.

Isso faria com que o texto "Exemplo" fosse exibido em vermelho.


Animações em CSS:

Animações CSS são uma maneira eficaz de adicionar movimento e interatividade aos elementos de uma página web,
permitindo que as pessoas desenvolvedoras criem experiências mais ricas e dinâmicas para as pessoas usuárias.


Melhores práticas para escrever um código CSS:

Organização do Código:

Manter o código CSS bem-organizado é essencial para facilitar a manutenção e a leitura.
Separe os códigos, use vários arquivos CSS para diferentes partes do site, ordene as propriedades de forma consistente (por ordem alfabética ou por categorias).


Responsividade:

Desenvolva seu CSS com foco em design responsivo, utilizando unidades relativas como porcentagens ou unidades de viewport, além de media queries.


Use folhas de estilo separadas para projetos maiores:

Isto não se aplica a todos os sites, mas em casos em que há um site grande com necessidade de um monte de CSS específicos, usar múltiplas folhas de estilo é uma boa ideia.


Evite a redundância:

Outra das melhores práticas do CSS a seguir é evitar a redundância, remova as repetições e seu código irá ler melhor e ter um melhor desempenho também.


Combine elementos onde for possível:

Ao invés de listar elementos um por um, combine-os para economizar espaço e tempo.
Muitas vezes, os elementos dentro de uma única folha de estilo terão o mesmo estilo (ou similar).
